📖 About FilmFusion

FilmFusion is a next-generation movie recommendation platform that uses AI (Google Gemini) to understand your emotions and provide personalized movie suggestions. Built with ASP.NET Core, PostgreSQL, and modern web technologies.

🎬 Features: Real movie data from TMDB | AI Chatbot | Favorites & Watchlist | User Ratings & Comments | Admin Dashboard
